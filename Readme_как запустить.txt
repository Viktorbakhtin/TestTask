1) Скачать проект по ссылке https://github.com/Viktorbakhtin/TestTask
2) Для локального сервера и работы с БД я использовал OpenServer (Настройки - > PHP 7.0, APACHE PHP-7.0, MySQL 5,6)
2.1) Нужно создать БД laravel_db, далее в самом проекте находим файл .env и выполняем настройки в этих строках 
{
DB_DATABASE=laravel_db
DB_USERNAME=root
DB_PASSWORD=
}

3) Далее заходим в консоль OpenServer и пишем 'php artisan migrate' (Это позволит в нашей бд создать еще пока что пустые таблицы с нужным названием)
4) Далее пишем в консоли php artisan db:seed (Это позволит заполнить наши таблицы некоторыми значениями)
5) Обязательно в openserver настройках->домены(указать имя домена и путь в папку public) 
6) Если все работает, переходим на наш сайт, который запущен на локальном сервере и смотрим тестовые задания по удобной навигации.
